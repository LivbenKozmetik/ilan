# Livben Kozmetik - Ürün İlanları

Bu proje, Livben Kozmetik firması için kozmetik ürün ilanlarını sergilemek amacıyla geliştirilmiş basit bir web uygulamasıdır.

## Özellikler

- 📱 Responsive tasarım (mobil uyumlu)
- 🎨 Modern ve şık arayüz
- 🛍️ Ürün kartları ile görsel sunum
- ✨ Smooth animasyonlar
- 🇹🇷 Türkçe dil desteği

## Dosya Yapısı

```
├── index.html      # Ana HTML dosyası
├── styles.css      # CSS stil dosyası  
├── script.js       # JavaScript işlevsellik
└── README.md       # Proje dokümantasyonu
```

## Kullanım

1. `index.html` dosyasını bir web tarayıcısında açın
2. Kozmetik ürünlerini keşfedin
3. "Detayları Gör" butonlarına tıklayarak ürün bilgilerini görüntüleyin

## Teknik Detaylar

- **HTML5** yapısında semantik etiketler
- **CSS3** ile modern stil ve animasyonlar
- **Vanilla JavaScript** ile interaktif özellikler
- Responsive grid yapısı
- Modern tarayıcılar ile uyumlu

## Geliştirme

Yeni ürün eklemek için `script.js` dosyasındaki `products` dizisini güncelleyin.
